# Introduction

Modern designed tailwindcss components that you can copy and paste into your apps. Accessible. Customizable. Open Source.

Ready to use components for:

- React.js
- Next.js
- Vue.js
- HTML / JS
- Others

### What makes MidUI different?

Midui is not any type of npm package, its just a collection of super awesome ready to use components which you can copy and paste into your web app saving time and work.

In addition to its other features, Midui provides comprehensive JavaScript logic that drives the functionality of its components. This ensures that each component operates seamlessly and efficiently, offering a robust solution for developers who require advanced interaction and behavior within their applications.
