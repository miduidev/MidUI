## Running Locally
`MIDUI ONLY WORKS ON LINUX ENVIRONMENT`

```sh
pnpm install
pnpm run dev

# To Build
pnpm install 
pnpm run build
```

## Why?
Because you need it to build modern website as fast as possible, saves time , less efforts , no need to install packages.
